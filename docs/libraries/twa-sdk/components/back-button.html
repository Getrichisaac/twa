BackButton

Controls the back button displayed in the header of the Web App in the Telegram interface. It is mostly used in case, when you want to provide a way to go back in routing history or rollback some action.

Initâ€‹

import {BackButton} from 'twa-sdk';  
import {init} from 'twa-bridge';  
  
const backButton = new BackButton();  
// or with your bridge instance.  
const backButton = new BackButton({bridge: init()});

Showing and hidingâ€‹

// Show back button.  
backButton.show();  
console.log(backButton.isVisible); // true  
  
// Hide back button.  
backButton.hide();  
console.log(backButton.isVisible); // false

Eventsâ€‹

Events available for listening:

  • visibleChange: (isVisible: boolean) => void
  • click: () => void

Methods supportâ€‹

Methods available for support check:

  • show
  • hide
